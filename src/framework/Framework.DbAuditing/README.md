# Catena-X Portal Backend Framework Auditing

The Catena-X Portal Backend Framework Auditing library provides a useful set of db auditing.

This content is produced and maintained by the Eclipse Tractus-X project.

* Project home: https://projects.eclipse.org/projects/automotive.tractusx

## Installation
